----모듈에는
- 프로젝트에 설치되는 모듈(프로젝트에서 cmd)
	웹소켓통신 : socket.io
		npm install socket.io
	

- Node.js본체에 설치되는 모듈(cmd로 하는거)
	프로젝트 만들기 편하게 : express
	실행하기 편하게 : nodemon
	cmd
		npm install express -g
		npm install express-generator@4 -g
		npm install nodemon -g

프로젝트 만들 폴더로 가서 cmd
	express 프로젝트명 	=> 프로젝트 생성
	cd 프로젝트명		=> 들어가서
	npm install		=> 프로젝트에서 사용하는 jar파일 설치
	nodemon 파일명		=> 실행하기(소스 수정해도 자동재시작)
		콘솔창 띄워놓기, 콘솔창 마우스 클릭해놓으면 실행안됨
